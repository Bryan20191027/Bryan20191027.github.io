2020/11/14 JAVA学习笔记 Sequence&&PrintStream

一.序列化与反序列化

1.java.io.ObjectOutputStream extends OutputStream 对象的序列化流

作用：把对象以流的方式写入到文件中

构造方法：

ObjectOutputStream(OutputStream out):创建写入指定OutputStream的ObjectOutputStream

特有的成员方法：

void writeObject(Object obj)将指定的对象写入ObjectOutputStream

使用步骤：

  1. 创建ObjectOutputStream对象，构造方法中传递字节输出流
  2. 使用ObjectOutputStream对象中的方法writeObject，把对象写入到文件中
  3. 释放资源

实例代码：

ArrayList<Person> list = new ArrayList<>();
list.add(new Person("ABC",15));
list.add(new Person("BCD",15));
list.add(new Person("FBI",15));
list.add(new Person("COM",15));
ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("list.txt"));
oos.writeObject(list);

注意事项：

  1. 如果抛出NotSerializableException没有序列化异常，则需要实现序列化接口
  2. 类通过实现java.io.Serializable接口以启用其序列化功能。未实现此接口的类将无法使其任何状态序列化或反序列化；Serializable接口也叫标记型接口，会给类添加一个标记
  3. 当我们进行序列化和反序列化的时候，就会检测类上是否有这个标记
  4. 还需要声明ClassNotFoundException

实例代码：

public class Person implements Serializable {
    static final long serialVersionUID = 42L;//显式声明序列号
    ...
}

2.java.io.ObjectInputStream extends InputStream

ObjectInputStream:对象中的反序列化流

作用：

把文件中保存的对象，以流的方式读取出来使用

构造方法：

ObjectInputStream(InputStream in)创建从指定InputStream读取的ObjectInputStream

特有的成员方法：

Object readObject() 从ObjectInputStream读取对象

使用步骤：

  1. 创建ObjectInputStream对象，构造方法中传递字节输入流
  2. 使用ObjectInputStream对象中的方法readObject读取保存对象的文件
  3. 释放资源
  4. 使用读取出来的对象（打印）

实例代码：

ObjectInputStream ois = new ObjectInputStream(new FileInputStream("person.txt"));
Object o = ois.readObject();
ois.close();
System.out.println(o);
ArrayList<Person> list2 = (ArrayList<Person>)o;
for(Person p:list2){
    System.out.println(p);
}

注意事项

反序列化前提：

  • 类必须实现Serializable接口
  • 必须存在类对应的Class文件

序列化异常


解决方法：显式的给类声明序列号

static final long serialVersionUID = 42L;

二.transient关键字：瞬态关键字

被它修饰的成员变量，不能被序列化，与static功能差不多，但是没有其静态含义

比较：

static关键字：静态关键字，静态优先于非静态加载到内存中（静态优先于对象进入到内存中）
被static修饰的成员变量不能被序列化，序列化的都是对象

三.java.io.PrintStream打印流

PrintStream为其他输出流添加了功能，使他们能够方便地打印各种数据值表示形式

PrintStream特点

  • 只负责数据的输出，不负责数据的读取
  • 永远不会抛出IOException
  • 有特有的方法，print，println

构造方法：

  • PrintStream(File file):输出的目的地是一个文件
  • PrintStream(OutputStream out):输出的目的地是一个字节输出流
  • PrintStream（String fileName):输出目的地是一个文件路径

注意事项：

  • 继承自OutputStream
  • 如果是用继承自父类的write方法写数据，那么查看数据的时候会查询编码表 97-》a
  • 如果使用自己特有的方法print/println，写的数据原样输出 97-》97
  • 可以改变输出语句的目的地（打印流的流向）
  • 输出语句，默认在控制台输出

   使用System.setOut方法改变输出语句的目的地改为参数中传递的打印流的目的地
   static void setOut(PrintStream out)重新分配标准输出流

实例代码：

PrintStream ps = new PrintStream("print.txt");
ps.write(97);//a
System.out.println("在控制台输出");
System.setOut(ps);
System.out.println("在打印流的目的地输出");
ps.println(97);
ps.close();
