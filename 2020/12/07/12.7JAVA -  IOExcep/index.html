2020/11/14 JAVA学习笔记 IOException&Properties

一.NullPointerException异常处理

JDK1.7之前：使用try catch finally处理流的异常

实例代码：

//提升fw的作用域让finally可以使用
    //变量再定义时可以没有值，但是使用时必须有值
    //fw = new FileWriter("e.txt",true);如果执行失败，没有值，fw.close会报错
    FileWriter fw = null; //用null来初始化
    try{
        fw = new FileWriter("e.txt",true);
        for(int i = 0;i<10;i++){
            fw.write("HelloWorld" + i + "\r\n");
        }
    }catch (IOException e){
        System.out.println(e);
    }finally {
        //创建对象失败，fw默认值为null,null不能调用方法，会抛出NullPointerException,需要增加一个判断，如果不是null再把资源释放
        try {
            //fw.close方法抛出了IOException异常对象,所以我们直接处理这个异常对象，要么throws，要么try  catch
            if(fw!=null)
                fw.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }

JDK1.7之后

在try后增加一个()，在括号中可以定义流对象，那么这个流对象的作用域就在try中有效，try中代码执行完毕会把流对象释放，不用写finally
格式:

try(定义流对象；定义流对象...)
    {可能会产生异常的代码}
catch(异常类变量 变量名）
    {异常的处理逻辑}

实例代码：

   try(FileInputStream fis = new FileInputStream("D:\\JAVA\\11.14.19.06IOStream\\b.txt");
       FileOutputStream fos = new FileOutputStream("a.txt");){
        int len = 0;
        while((len = fis.read()) !=-1){
            fos.write(len);
        }
    }
    catch(IOException e){
        System.out.println(e);
    }

二.Properties

java.util.Properties集合 extends Hashtable<k,v> implements Map<k,v>

1.定义

Properties类表示一个持久的属性值，Properties可保存在流中或从流中加载，他是唯一一个与IO流相结合的集合
注意事项：

  • 属性列表中每个键及其对应值都是一个字符串
  • Properties集合是一个双列集合，key和value默认都是字符串

2.方法

（1）store，把集合中的临时数据持久化写入到硬盘中存储

注意：

  1. 存储键值对的文件中，键与值默认的连接符号可以使用=，空格(其他符号）
  2. 存储键值对的文件中，可以使用进行注释，被注释的键值对不会再被读取
  3. 存储键值对的文件中，键与值默认都是字符串，不用再加引号

（2）load，把硬盘中保存的文件(键值对)，读取到集合中使用

  • void load(InputStream inStream)
  • void load(Reader reader)

参数：

  • InputStream inStream:字节输入流，不能读取含有中文的键值对
  • Reader reader:字符输入流，能读取含有中文的键值对

使用步骤：

  1. 创建Properties集合对象
  2. 使用Properties集合对象中的方法load读取保存键值对的文件
  3. 遍历Properties集合

3.特有方法

使用properties集合存储数据，遍历取出Properties集合中的数据
特有方法：

  • Object setProperty(String key,String value)调用hashtable中的方法put
  • String getProperty(String key),通过key找到value值，此方法相当于Map集合中的get(key)方法
  • Set stringPropertyNames() 返回此属性列表中的键集，其中该键及其对应值是字符串

实例代码：

//getProperty()&stringPropertyNames();
Properties prop = new Properties();
    prop.load(new FileReader("b.txt"));
    Set<String> strings = prop.stringPropertyNames();
    for(String key:strings){
        String value = prop.getProperty(key);
        System.out.println(key + "=" +value);
    }

//setProperty()
Properties prop = new Properties();
    prop.setProperty("zhao","198");
    prop.setProperty("di","167");
    prop.setProperty("wu","189");
    for(String name: prop.stringPropertyNames()){
        System.out.println(name + "" + prop.getProperty(name));
    }
