2020/11/17 JAVA学习笔记 JDBC

Derby数据库

使用步骤:

  1. 启动ij环境

  2. 连接内置Derby数据库并创建表

    （创建并连接内置Derby数据库）
    create table 表名(字段1 属性, 字段2 属性…字段n 属性);

  3. 向表中插入行&&查询表中记录
    一次插入一条记录：insert into 表名 values(字段1值, 字段2值, … , 字段n值);
    一次插入多条记录：insert into 表名 values(字段1值, 字段2值, … , 字段n值), (字段1值, 字段2值, … ,字段n值) … ;

    记录带着全部字段值：select * from 表名;
    带着部分字段值：select 字段m, … , 字段n from 表名;

  4. 更新表中的记录
    update <表名> set <字段名>=新值 where <条件子句>;

  5. 删除表中记录
    delete from <表名> where <条件子句>;

  6. 在数据库中删除表
    drop table 表名;
