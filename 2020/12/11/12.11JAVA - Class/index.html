2020/11/17 JAVA学习笔记 Class

一.Class类

定义：

  • 类结构的描述，每一个类在加载的时候，虚拟机都会根据加载的类创建对应的Class对象
  • Class对象是具体类的描述，包含类名、属性、方法、构造方法等，在内存的程序区

获取Class对象的方法：

  • Class.forName(“类全限定名”)，在动态加载一个类的时候使用，因为类名是在运行时确定的
    需要throws ClassNotFoundException
  • 类名.class:返回类名，类名在程序编译时确定
  • 对象名.getClass():返回实例的类型，对实例化对象调用，该方法在Object中定义

实例代码：

System.out.println(Class.forName("Person"));
System.out.println(Person.class);

Person p1 = new Person(12,"no");
System.out.println(p1.getClass());
//输出均为class Person

二.创建（实例化）对象的方式

  • 用new语句创建对象(最常见）

  • 通过工厂方法返回对象

      String str = String.valueOf(23);

  • 运用反射手段,调用java.lang.Class或者java.lang.reflect.Constructor类的newInstance()实例方法

      Object obj = Class.forName("java.lang.Object").newInstance(); 
  //需要throws ClassNotFoundException, IllegalAccessException, InstantiationException
  Object obj = Class.forName("Person").newInstance();
  System.out.println(obj.getClass());//输出class Person，而非Object

  • 调用对象的clone()方法。
    需要类implements Cloneable，并且实现clone()方法

      //部分实例代码：
  //Person类中的clone方法：
  public class Person implements Cloneable{
      //...
      public Object clone(){
          Person o = null;
          try{
              o = (Person)super.clone();
          }catch(CloneNotSupportedException e){
              e.printStackTrace();
          }
          return o;
      }
      //...
  }

  //clone()使用，需要转型
  Person p2 = (Person) p1.clone();

  • 通过I/O流（包括反序列化），如运用反序列化手段，调用java.io.ObjectInputStream对象的 readObject()方法。

类的加载

  1. JVM对于每一个要使用的类首先要用类装载器将类的字节码数据装载
  2. 类装载器根据类名来定位和装载字节码数据，返回给JVM
  3. 类加载器将.class文件内容转化为JVM能接收的字节码数据
  4. JVM将字节码数据编译为可运行代码存储载内存，并索引保存在HashTable

Java.lang.ClassLoader基类的loadClass方法加载类：

  • 从本地文件系统加载class文件
  • 从jar加载class文件
  • 通过网络加载class文件
  • 类加载器本身通过java内嵌的Bootstrap的装载

static块

  • Class.forName()加载后再调用static块进行静态初始化
  • 类名.class只对类加载，不调用static块，也不对静态变量初始化

实例代码：

static char sex= 'F';
static int select;
static{
    if(sex == 'F'){
        select = 1;
    }else{
        select = 2;
    }
}
//所使用变量均为静态
