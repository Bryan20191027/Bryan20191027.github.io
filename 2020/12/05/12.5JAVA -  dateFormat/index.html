2020/11/12 JAVA学习笔记 DateFormat类

DateFormat类

1.定义：

是日期/时间格式化子类的抽象类，我们可以通过这个类可以帮我们完成日期和文本之间的转换；
也就是可以在Date对象与String对象之间进行来回转换

格式化：从Date对象转换为String对象
解析：按照指定的格式，从String对象转换为Date对象
成员方法：

  String format(Date date):格式化  
  Date parse(String source):解析

该类为抽象类，无法直接使用，可以使用子类

2.使用子类：SimpleDateFormat

java.text.SimpleDateFormat extends DateFormat

构造方法：

SimpleDateFormat(String pattern)
//用给定的模式和默认语言环境的日期格式符号构造

参数介绍：
String pattern:传递指定的模式
模式：区分大小写:y 年 M月 d日 H时 m分 s秒
注意事项：字母不能更改，连接模式的符号可以改变

使用步骤：

1.创建SimpleDateFormat对象，构造方法中传递指定的模式
2.调用SimpleDateFormat对象中的方法format

实例代码：

private static void formatTime(){
    SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
    Date date = new Date();
    String text = sdf.format(date);
}

parse方法

用途：把符合模式的字符串解析为Date日期
1.创建SimpleDateFormat对象，构造方法中传递指定的模式
2.调用SimpleDateFormat对象中的方法parse
注意： 

public Date parse(String source) throws ParseException

parse方法声明了一个叫ParseException解析异常的异常
如果字符串和构造方法中的模式不一样，那么程序就会抛出此异常
所以方法声明中需要写throws ParseException

实例代码：

private static void parseTime(String str) throws ParseException {
    SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
    System.out.println(sdf.parse(str));
}
