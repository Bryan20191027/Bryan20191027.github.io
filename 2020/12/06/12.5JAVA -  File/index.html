2020/11/13 JAVA学习笔记 File

一.File类

1.路径：

  • 绝对路径：是一个完整的路径，以盘符开始的路径
  • 相对路径：是一个简化的路径，
  • 相对指的是相对于当前项目的根目录，如果使用当前项目的根目录，路径可以简化书写
    注意事项：

1.windows路径是不区分大小写的
2.反斜杠是转义字符，两个反斜杠代表一个普通的反斜杠

获取分隔符

实例代码：

String pathSeparator = File.pathSeparator;
System.out.println(pathSeparator);//路径分隔符 windows：分号;  linux：冒号：

String separator = File.separator;
System.out.println(separator);//文件名称分隔符 windows：反斜杠\ linux：正斜杠/

//路径可以写成"C:"+File.separator+"Users"+File.separator+"86180"+File.separato+"Desktop"
//即不需要写分隔符

2.构造方法

File(String name)通过将给定路径名字符串转换为抽象路径来创建一个新File实例
参数：String pathname：字符串的路径名称；

  • 可以以文件，文件夹结尾；
  • 可以是相对路径和绝对路径
  • 可以存在，也可以不存在
  • 创建File对象，只是把字符串路径封装为File对象，不考虑路径的真假情况

实例代码：

File f1 = new File("C:\\Users\\86180\\Desktop\\a.txt");

File(File parent,String child)根据parent路径名字符串和child路径名字符串创建一个新File实例
参数：把路径分成了两部分

  • File parent:父路径
  • String child:子路径

父路径是File类型，可以使用File的方法对路径进行操作，在使用路径创建对象

实例代码：

File f1 = new File("C:\\Users\\86180\\Desktop");
File file = new File(f1,"a.txt");

File(String parent,String child)根据parent路径名字符串和child路径名字符串创建一个新File实例
参数：把路径分成了两部分

  • String parent:父路径
  • String child:子路径

好处：可以单独书写用起来灵活，都可以变化

实例代码：

File f1 = new File("C:\\Users\\86180\\Desktop","a.txt");

3.获取功能方法

  • public String getAbsolutePath():返回绝对路径名字符串
  • public String getPath():将此File转换为路径名字符串
  • public String getName():返回由此File表示的文件或目录的名称
  • public long length()：返回由此File表示的文件长度

实例代码：

//getAbsolutePath()
String absolutePath = f1.getAbsolutePath();
System.out.println(absolutePath);//输出C:\Users\86180\Desktop\a.txt

//getPath()
File f1 = new File("C:\\Users\\86180\\Desktop\\a.txt");
String Path = f1.getPath();
System.out.println(Path);//C:\Users\86180\Desktop\a.txt

File f2 = new File("a.txt");
System.out.println(f2.getPath());

System.out.println(f1);//C:\Users\86180\Desktop\a.txt
System.out.println(f1.toString());//C:\Users\86180\Desktop\a.txt

//getName()
File f1 = new File("C:\\Users\\86180\\Desktop\\a.txt");
System.out.println(f1.getName());//输出a.txt

//length()
File f1 = new File("C:\\Users\\86180\\Desktop\\a.txt");
System.out.println(f1.length());
//获取的是构造方法指定的文件大小，以字节为单位
//注意：文件夹没有大小概念，不能获取文件夹的大小
//如果路径不存在返回0

4.判断功能方法

  • public boolean exists():此File表示的文件或目录是否存在
  • public boolean isDirectory()：此File表示的是否为目录
  • public boolean isFile():表示的是否为文件

5.创建与删除方法

public boolean createNewFile():当且仅当具有该名称的文件不存在时，创建新空文件
返回值：true文件不存在，创建文件；false：文件存在，不创建文件,不存在该路径返回false
注意：

  • 只能创建文件，不能创建文件夹
  • 创建文件路径必须存在，否则抛出异常

实例代码：

File f1 = new File("C:\\Users\\86180\\Desktop","b.txt");
boolean b1 = f1.createNewFile();
File f2 = new File("C:\\Users\\86180\\Desktop\\文件夹");//并非创建了文件夹
boolean b2 = f1.createNewFile();

public boolean mkdir():创建由此File表示的目录
public boolean mkdirs():创建由此File表示目录，包括任何必需但不存在的父目录
注意：mkdir创建单级文件夹，mkdirs创建多级文件夹

实例代码：

File f2 = new File("C:\\Users\\86180\\Desktop\\文件夹\\");
//File f2 = new File("C:\\Users\\86180\\Desktop\\文件夹\\111\\111\\222\\333");
//只能创建单级文件夹，返回false
System.out.println(f2.mkdir());

File f1 = new File("C:\\Users\\86180\\Desktop\\文件夹\\111\\111\\222\\333");

public boolean delete():删除由此File表示的文件或目录
此方法可以删除构造方法路径中给出的文件/文件夹
返回值：

  • true：文件/文件夹删除成功
  • false：文件夹中有内容，不会删除；构造路径不存在；

注意：delete方法是直接在硬盘删除文件/文件夹，不走回收站，删除要谨慎

6.遍历

public String[] list():返回一个String数组表示该File目录中的所有子文件或目录
遍历构造方法中给出的目录，会获取目录中所有文件/文件夹的名称，表示该File目录中的所有子文件或目录
会打印隐藏文件/文件夹

实例代码：

File file = new File("D:\\CloudMusic");
String[] arr = file.list();
for(String str:arr){
    System.out.println(str);
}

public File[] listFiles():返回一个File数组表示该File目录中的所有子文件或目录
遍历构造方法中给出的目录，会获取目录中所有文件/文件夹的名称，把文件/文件夹封装为File对象，存储到File数组中

实例代码：

File file = new File("D:\\CloudMusic");
File[] arr = file.listFiles();
for(File filename:arr){
    System.out.println(filename);//打印绝对路径
}

注意事项：遍历的是构造方法中给出的目录

  • 如果构造方法中给出目录路径不存在，会抛出空指针异常
  • 如果构造方法中给出路径不是一个目录，也会抛出空指针异常
