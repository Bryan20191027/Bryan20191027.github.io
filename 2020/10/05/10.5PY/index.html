机器学习100天——Day1 数据预处理

转载自Github原教程网址

第一步：导入相关库

import numpy as np
import pandas as pd

第二步：导入数据集

dataset = pd.read_csv('Data.csv')    #读取csv文件
X = dataset.iloc[ : , :-1].values    #.iloc[行，列]，共有四列，取到了索引值为0~2的前三列
Y = dataset.iloc[ : , 3].values    #只取了索引值为3的第四列

第三步：处理丢失数据

from sklearn.impute import SimpleImputer
#注意点1：原文中为from sklearn.preprocessing import Imputer，但是版本更新后不可使用
imputer = SimpleImputer(missing_values = np.nan,strategy = "mean")    
#注意点2：原文中参数列表中还有一个参数为axis = 0，但已被弃用
#注意点3：原文中missing_values后输入值为"NaN"
imputer = imputer.fit(X[ : , 1:3])
X[ : , 1:3] = imputer.transform(X[ : , 1:3])

SimpleImputer的missing_values为缺失值。
strategy就是你采取什么样的策略去填充空值，总共有4种选择。分别是mean,median, most_frequent,以及constant。对于每一列来说的，如果是mean，则该列则由该列的均值填充。而median,则是中位数，most_frequent则是众数。
需要注意的是，如果是constant,则可以将空值填充为自定义的值，这就要涉及到后面一个参数了，也就是fill_value。如果strategy=’constant’,则填充fill_value的值。
