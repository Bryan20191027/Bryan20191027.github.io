2020/10/8 算法学习笔记

10.8 算法学习笔记

1.基础

(1)基础输入输出语句

scanf("%d%d",&a,&b);
printf("%.1f\n,a+b);
//.1f代表着浮点数保留一位小数输出
//04d表示小数点前至少保留4位有效数字，如果不够位则用0补
//4d则不会使用0补

(2)短路策略：
a为真，b无论是如何,a||b均为真，不会进行b中的运算

(3)转义：
\\n，编译器会把双斜线\\理解为\

(4)floor(x)函数返回不超过x的最大整数

(5)管道小技巧：在windows命令行下执行echo 20|abc，操作系统会自动把20输入，其中abc是程序名。

(6)使用scanf(“%d”,&x) == 1进行判断，返回的是成功输入的变量个数，当输入结束时scanf函数无法读取x，返回0.按Enter不意味着输入的结束，在windows下，输入完毕后先按Enter键，再按Ctrl+Z键，最后再按Enter键，即可结束输入。

(7)变量在未赋值之前值不确定，特别地他不一定等于0. 

#define INF = 100000000
max = -INF；
min = INF;

(8)输入输出重定向：

freopen("input.txt","r",stdin);
freopen("output.txt","w",stdout);
//将使得scanf从文件input.txt读入，printf写入文件output.txt。所用读键盘输入、写屏幕输入的函数都将改用文件

(9)要求用文件输入输出（非重定向方式）

#include<stdio.h>
#define INF 1000000000
int main()
{
    FILE *fin,*fout;
    fin = fopen("data.in","rb");
    fout = fopen("data.out","wb");
    int x,n = 0,min = INF,max = -INF,s = 0;
    while(fscanf(fin,"%d",&x)==1)
    {
        s+=x;
        if(x<min)min = x;
        if(x>max)max = x;
        n++;
    }
    fprintf(fout,"%d %d %.3f\n",min,max,(double)s/n);
    fclose(fin);
    fclose(fout);
    return 0;
}

2.竞赛注意事项

(1)输入输出格式规定：
不要在用户输入之前打印提示信息（例如”Please input:”）
(2)不要让程序按任意键退出（例如调用system(“pause”)，或者调用多余getchar()），输出完毕后立即终止程序，不需要等待用户按键。
(3)不要在算法竞赛中使用getch()、getcher()、gotoxy()、clrscr()函数
(4)KISS(Keep It Simple and Stupid)原则
(5)竞赛一般使用的是int都是32位整数
